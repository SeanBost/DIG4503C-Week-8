## Project Description/Plan
 - Create a webapp that will let users enter in their commute times for the week (same commute for each week assumed for simplicity) and then let them know what weather to expect when out on the road in a way that highlights useful information (rain, extreme temp). Will utilize free gov weather API.

 ## Features
  - Let users input and change their commute days/times (key feature)
  - Save commute information to localStorage (data persistence)
  - Display upcoming weather for their commutes this week (key feature)
  - Fully responsive layout (usability/rubric feature)
  - Deployed to Vercel (rubric feature)
  - Let users adjust their commute data (bonus/non-MVP feature)
  
  ## Tech Stack
  - React, location to long/lat API, weather API, Vercel
